This folder contains the basic ways to address Video Indexer's API in order to allow full engagment with the product: upload a video, polling on status/waiting for url callback,
indexing the video, getting the video/player widgets. A more detailed documntation can be found [here](https://docs.microsoft.com/en-us/azure/media-services/video-indexer/video-indexer-use-apis).

For more API abilities, please visit our [API documentation](https://api-portal.videoindexer.ai/docs/services/)

## Contents

Contains samples code for Video Indexer API and Widgets

| File/folder                       | Description                                    |
|-----------------------------------|------------------------------------------------|
| `upload-video`                    | Upload video to Video Indexer                  |
| `status-polling`                  | Polling on status change or using url callback |
| `get-insights-player-widgets`     | Get the player/insights widget                 |
| `customize-face-model`            | Customization of the face recognition model    |
| `customize-language-model`        | Customization of the language model            |



## Prerequisites

Valid Video Indexer account. For more information on how to do it please start from the [main Video Indexer GitHub page](https://github.com/itayar/media-services-video-indexer).
